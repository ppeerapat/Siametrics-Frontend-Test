Make sure to run both backend and frontend

### Backend

1. Installation

```
npm install -g json-server
```

2. Start a backend server

```
yarn start
```

or

```
json-server --watch db.json
```

### Frontend

1. Navigate to folder /front
2. Install packages

```
yarn install
```

3. Start a frontend server

```
yarn start
```

4. Head to
   `http://localhost:3030/
